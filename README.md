A comprehensive machine learning repository that investigates the theoretical and practical aspects of multiclass and ordinal classification through a real-world Heart Disease Severity Prediction problem. The project emphasizes systematic model development, rigorous evaluation, explainable artificial intelligence (XAI), and reproducible deployment practices.
Unlike conventional classification projects that primarily optimize predictive accuracy, this repository explores how the underlying structure of target labels influences model selection, optimization strategies, and evaluation methodology. Special attention is given to distinguishing nominal and ordinal classification paradigms, demonstrating how ordered target variables require fundamentally different performance assessment compared to traditional multiclass problems.
The repository presents a comparative analysis of classical machine learning algorithms, ensemble learning techniques, gradient boosting frameworks, and ordinal learning approaches under a unified experimental framework. Model performance is evaluated using both conventional classification metrics and Quadratic Weighted Kappa (QWK) to quantify prediction quality while respecting ordinal class relationships. Furthermore, model interpretability is investigated through SHAP (SHapley Additive Explanations) to analyze feature contributions at both global and local levels.
This project follows an end-to-end machine learning workflow encompassing exploratory data analysis, feature engineering, model development, hyperparameter optimization, statistical performance evaluation, explainable AI, and production-ready deployment using Flask.

Research Objectives
1)Investigate the behavioral differences between nominal and ordinal classification problems.
2)Benchmark multiple machine learning algorithms under identical experimental conditions.
3)Analyze the limitations of conventional evaluation metrics for ordered target prediction.
4)Evaluate the effectiveness of Quadratic Weighted Kappa (QWK) for ordinal classification.
5)Interpret model predictions using SHAP-based explainability techniques.
6)Develop a reproducible and deployable machine learning pipeline following industry-standard practices.
